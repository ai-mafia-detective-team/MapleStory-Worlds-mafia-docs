# 루디브리엄 시간수사대

## MapleStory Worlds 기반 멀티 추리 마피아 월드 기획안

---

## 1. 프로젝트 개요

**루디브리엄 시간수사대**는 MapleStory Worlds 플랫폼에서 제작되는 멀티 추리 마피아 파티게임이다.

플레이어는 루디브리엄 시간탑에 모인 수사대원이 되어, 마을에 숨어든 배신자를 찾아내야 한다. 시민팀은 토론, 투표, 직업 능력, 단서 추적을 통해 배신자를 색출하고, 배신자팀은 정체를 숨긴 채 시민팀을 혼란에 빠뜨려 승리를 노린다.

본 월드는 LLM이나 AI NPC에 의존하지 않고, 사람 간의 토론, 심리전, 추리, 투표를 핵심 재미로 삼는다. 메이플스토리의 마을, 직업, 몬스터, 시간탑 콘셉트를 마피아 장르와 결합하여 기존 메이플 IP를 새로운 멀티 파티게임 방식으로 확장하는 것을 목표로 한다.

---

## 2. 기획 배경

마피아 게임은 짧은 플레이 시간 안에서 강한 몰입감, 반복성, 소셜 재미를 제공하는 장르다. 그러나 일반적인 마피아 게임은 배경과 역할 구성이 단순해 차별화가 어렵다.

MapleStory Worlds는 메이플스토리 IP의 친숙한 캐릭터, 맵, 몬스터, 직업 콘셉트를 활용할 수 있는 플랫폼이다. 이 장점을 살려 단순한 마피아 게임이 아니라, 메이플 세계관 속에서 발생한 사건을 해결하는 “수사형 멀티 추리 월드”로 재해석하고자 한다.

특히 루디브리엄은 시간, 장난감, 시계탑이라는 강한 테마를 가지고 있어 “시간을 조작하는 배신자”, “기록을 복원하는 수사대”, “라운드마다 반복되는 사건” 같은 추리 게임 장치와 자연스럽게 결합된다.

---

## 3. 게임 콘셉트

### 핵심 콘셉트

> 루디브리엄 시간탑의 시간이 멈췄다.
> 수사대원들 사이에 숨어든 배신자를 찾아내지 못하면 시간탑은 붕괴된다.

플레이어는 매 라운드 낮과 밤을 반복하며 단서를 모으고, 토론을 통해 의심 대상을 좁힌다. 밤에는 각 직업의 능력이 발동되며, 낮에는 사건 결과와 플레이어들의 발언을 바탕으로 투표가 진행된다.

---

## 4. 장르 및 플레이 방식

| 구분        | 내용                          |
| --------- | --------------------------- |
| 장르        | 멀티 추리 / 마피아 / 파티게임          |
| 플랫폼       | MapleStory Worlds           |
| 플레이 인원    | 4~8명                        |
| 예상 플레이 시간 | 1판 7~12분                    |
| 진행 방식     | 낮 토론 → 투표 → 밤 행동 → 결과 공개 반복 |
| 핵심 재미     | 추리, 심리전, 거짓말, 협동, 경쟁        |
| AI 사용 여부  | 게임 플레이 내 LLM/AI 미사용         |

---

## 5. 세계관

루디브리엄 시간탑의 시계가 갑자기 멈추고, 마을 곳곳에서 이상 현상이 발생한다. 시간탑을 관리하던 수리공들은 이 사건이 단순 고장이 아니라 내부자의 조작이라는 사실을 알게 된다.

플레이어들은 시간수사대가 되어 사건의 원인을 조사하지만, 수사대 안에는 시간탑을 파괴하려는 배신자가 숨어 있다. 배신자는 밤마다 시민팀을 방해하고, 기록을 조작하며, 자신에게 향하는 의심을 다른 플레이어에게 돌린다.

시민팀은 배신자를 찾아 시간탑을 복구해야 하고, 배신자팀은 정체를 숨긴 채 수사대를 무너뜨려야 한다.

---

## 6. 핵심 게임 루프

```txt
로비 입장
→ 방 생성 또는 참가
→ 플레이어 준비 완료
→ 역할 자동 배정
→ 낮 토론
→ 투표
→ 밤 직업 능력 사용
→ 사건 결과 공개
→ 승패 조건 확인
→ 다음 라운드 진행
→ 결과 및 보상 지급
```

이 구조를 통해 플레이어는 매 판 다른 역할, 다른 의심 구도, 다른 투표 결과를 경험하게 된다.

---

## 7. 진영 구성

### 시민팀

시민팀은 배신자를 찾아내는 것을 목표로 한다.
토론, 투표, 직업 능력, 단서 확인을 통해 배신자팀을 제거하면 승리한다.

### 배신자팀

배신자팀은 시민팀의 수를 줄이거나, 수사 과정을 방해하는 것을 목표로 한다.
정체를 숨기고 거짓 정보를 퍼뜨리며, 시민팀의 투표를 잘못된 방향으로 유도한다.

---

## 8. 직업 설계

### 시민팀 직업

| 직업     | 설명                | 능력                      |
| ------ | ----------------- | ----------------------- |
| 시간수리공  | 시간탑을 복구하는 핵심 수사대원 | 밤에 한 명을 보호한다            |
| 기록관    | 시간탑의 기록을 읽는 조사관   | 이전 라운드 투표 기록 일부를 확인한다   |
| 예언자    | 수상한 기운을 감지하는 마법사  | 한 명의 진영을 확인한다           |
| 장난감 병정 | 루디브리엄을 지키는 경비병    | 특정 플레이어를 밤 공격에서 지킨다     |
| 모험가    | 기본 시민 역할          | 특별 능력은 없지만 토론과 투표로 기여한다 |

### 배신자팀 직업

| 직업      | 설명                | 능력                    |
| ------- | ----------------- | --------------------- |
| 시간도둑    | 시간탑의 흐름을 조작하는 배신자 | 한 명의 행동 기록을 흐리게 만든다   |
| 변장 몬스터  | 시민으로 위장한 몬스터      | 조사 결과를 시민처럼 보이게 한다    |
| 시계탑 파괴자 | 시간탑을 붕괴시키려는 적     | 특정 구역 또는 기능을 한 턴 방해한다 |

MVP에서는 직업을 6개 내외로 제한하고, 이후 업데이트를 통해 신규 직업을 추가한다.

| ------ | --------------------------------------- |

## 10. 주요 시스템

### 10.1 방 생성 및 참가 시스템

플레이어는 로비에서 방을 생성하거나 기존 방에 참가할 수 있다.
방장은 인원 수, 토론 시간, 투표 시간, 직업 포함 여부를 설정할 수 있다.

### 10.2 역할 배정 시스템

게임 시작 시 서버 로직을 통해 각 플레이어에게 역할이 무작위로 배정된다.
역할 정보는 본인에게만 공개되며, 다른 플레이어는 알 수 없다.

### 10.3 페이즈 시스템

게임은 낮, 투표, 밤, 결과 공개 페이즈로 구성된다.

| 페이즈 | 내용                  |
| --- | ------------------- |
| 낮   | 토론, 단서 공유, 의심 대상 추리 |
| 투표  | 추방할 플레이어 선택         |
| 밤   | 직업 능력 사용            |
| 결과  | 밤 행동 및 투표 결과 공개     |

### 10.4 투표 시스템

생존 플레이어는 투표 시간 동안 한 명을 선택하거나 건너뛸 수 있다.
최다 득표자는 추방되며, 동률일 경우 추방 없음 또는 재투표 규칙을 적용한다.

### 10.5 직업 능력 시스템

각 직업은 밤에 능력을 사용할 수 있다.
능력은 보호, 조사, 기록 확인, 방해, 조작 등으로 구성된다.

### 10.6 전적 및 보상 시스템

DataStorage를 활용하여 플레이어의 누적 전적을 저장한다.

저장 항목은 다음과 같다.

| 항목      | 설명            |
| ------- | ------------- |
| 플레이 횟수  | 총 게임 참여 횟수    |
| 승리 횟수   | 전체 승리 수       |
| 시민팀 승리  | 시민팀으로 승리한 횟수  |
| 배신자팀 승리 | 배신자팀으로 승리한 횟수 |
| 추리 성공   | 배신자 투표 적중 횟수  |
| 칭호      | 조건 달성 시 획득    |
| 시즌 포인트  | 시즌 랭킹에 반영     |

## 12. 메이플 IP 활용 방식

본 월드는 메이플스토리 IP를 단순 배경 장식으로 사용하는 것이 아니라, 게임 규칙과 세계관에 직접 연결한다.

| IP 요소   | 활용 방식             |
| ------- | ----------------- |
| 루디브리엄   | 메인 배경             |
| 시간탑     | 라운드 반복, 시간 조작 콘셉트 |
| 장난감 병정  | 시민팀 직업            |
| 몬스터     | 배신자팀 콘셉트          |
| 모험가 직업군 | 시민팀 역할 구조         |
| 퀘스트 감성  | 사건 해결형 진행         |
| 파티 플레이  | 멀티 토론/투표 구조       |

이를 통해 메이플스토리의 친숙한 IP를 추리형 멀티 파티게임으로 확장한다.

## 13. 차별화 포인트

### 13.1 메이플 IP 기반 마피아

기존 마피아 게임은 현실 배경이나 추상적인 역할에 의존하는 경우가 많다.
본 월드는 루디브리엄, 시간탑, 몬스터, 직업군을 활용하여 메이플스토리 세계관 안에서 자연스럽게 마피아 룰을 재구성한다.

### 13.2 시간 조작 콘셉트

루디브리엄의 시간 테마를 활용하여 기록 확인, 행동 조작, 보호, 복구 같은 능력을 설계한다.
이는 일반적인 마피아 게임과 다른 전략적 재미를 제공한다.

### 13.3 짧고 반복 가능한 플레이

1판 7~12분의 짧은 플레이 타임을 목표로 하여 친구들과 부담 없이 반복 플레이할 수 있다.

### 13.4 장기 운영 가능성

전적, 칭호, 시즌 랭킹, 신규 직업, 신규 맵을 통해 출시 이후 지속적인 업데이트가 가능하다.

## 17. 수익화 방향

본 월드는 게임 밸런스에 영향을 주는 유료 요소를 배제한다.
능력 강화, 특정 직업 확정, 배신자 확률 증가 같은 Pay-to-Win 요소는 도입하지 않는다.

수익화는 꾸미기 중심으로 설계한다.

| 판매 가능 요소  | 설명                |
| --------- | ----------------- |
| 말풍선 스킨    | 토론 중 표시되는 말풍선 꾸미기 |
| 투표 이펙트    | 투표 시 연출 효과        |
| 승리 포즈     | 결과 화면에서 표시        |
| 칭호 프레임    | 닉네임 주변 장식         |
| 로비 감정표현   | 대기 중 사용하는 이모션     |
| 시즌 꾸미기 보상 | 시즌 플레이 보상         |

## 18. 기대 효과

**루디브리엄 시간수사대**는 메이플스토리의 친숙한 IP를 활용하면서도, 기존 RPG 중심 월드와 다른 소셜 추리 장르의 재미를 제공한다.

짧은 플레이 타임, 반복 가능한 마피아 룰, 직업별 능력, 시즌 업데이트 구조를 통해 유저가 지속적으로 재방문할 수 있는 월드를 목표로 한다.

또한 DataStorage 기반 전적과 칭호, 시즌 랭킹을 통해 단발성 미니게임이 아니라 장기 운영 가능한 멀티 파티게임으로 확장할 수 있다.
