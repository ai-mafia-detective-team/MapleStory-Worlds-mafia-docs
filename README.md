# MapleStory Worlds Mafia Docs

> **커닝시티: 지하수사록**
> MapleStory Worlds 기반 멀티 추리 마피아 게임 기획·개발 문서 저장소

## 프로젝트 소개

**커닝시티: 지하수사록**은 MapleStory Worlds에서 제작하는 4~8인 멀티 추리 마피아 게임입니다.
플레이어는 커닝시티 지하 수사대의 일원이 되어, 시민 사이에 숨어든 마피아를 찾아내야 합니다.

낮에는 토론과 투표를 통해 의심 대상을 추리하고, 밤에는 각 직업의 능력을 사용해 정보를 얻거나 사건을 조작합니다.
메이플스토리 특유의 분위기와 커닝시티의 어두운 지하 세계를 바탕으로, 캐주얼하지만 긴장감 있는 추리 경험을 목표로 합니다.

---

## 핵심 콘셉트

* **장르**: 멀티플레이 추리 / 마피아 / 소셜 디덕션
* **플랫폼**: MapleStory Worlds
* **플레이 인원**: 4~8명
* **배경**: 커닝시티 지하 수사 구역
* **주요 흐름**: 로비 → 방 생성/입장 → 역할 배정 → 낮 토론 → 투표 → 밤 행동 → 결과 공개 → 승패 판정

---

## 게임 시나리오 요약

커닝시티 지하에서 의문의 사건들이 연달아 발생한다.
정보상 실종, 비밀 장부 도난, 지하 폭발 사고, 그리고 수사대 내부에 숨어든 배신자에 대한 소문.

플레이어들은 익명의 초대장을 받고 지하 집회장에 모인다.
하지만 문이 닫히는 순간, 이곳에 모인 사람들 중 누군가가 이미 마피아라는 사실을 깨닫게 된다.

수사대는 제한된 시간 안에 마피아를 찾아내야 한다.

## 주요 기능

### 1. 로비

* 플레이어 캐릭터 표시
* 닉네임 표시
* 방 만들기
* 수사방 찾기
* 빠른 시작
* 친구 / 알림 / 설정 / 상점 / 프로필 메뉴

### 2. 방 시스템

* 공개방 / 비공개방
* 인원 제한
* 방 제목 설정
* 방장 표시
* 입장 상태 표시
* 페이지네이션 방식의 방 목록

### 3. 게임 진행

* 역할 랜덤 배정
* 낮 토론 시간
* 투표 시스템
* 밤 행동 시스템
* 생존 / 사망 상태 관리
* 결과 공개
* 승리 조건 판정

### 4. 역할 시스템

초기 MVP 기준 역할은 다음과 같이 구성합니다.

| 진영  | 역할  | 설명                        |
| --- | --- | ------------------------- |
| 시민  | 시민  | 토론과 투표로 마피아를 찾아야 하는 기본 역할 |
| 시민  | 경찰  | 밤마다 한 명의 정체를 조사할 수 있음     |
| 시민  | 의사  | 밤마다 한 명을 보호할 수 있음         |
| 마피아 | 마피아 | 밤마다 한 명을 제거할 수 있음         |

추후 밸런스에 따라 특수 직업을 추가할 수 있습니다.
