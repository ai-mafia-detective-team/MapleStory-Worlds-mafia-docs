# AI 마피아 수사대

## 1. 프로젝트 개요

**AI 마피아 수사대**는 사람과 AI가 함께 플레이할 수 있는 하이브리드 마피아 게임 앱입니다.

사용자는 모바일 앱에서 게임방을 생성하거나 빠른 시작을 통해 게임에 참여할 수 있으며, 사람끼리만 플레이하거나 AI를 포함해 플레이할 수 있습니다. 부족한 인원은 AI가 자동으로 채워질 수 있고, 마피아와 직업은 사람과 AI 구분 없이 랜덤으로 배정됩니다.

AI 플레이어는 LLM 기반으로 한국어 대화를 수행하며, 자신의 직업과 성격, 비밀 정보에 따라 진실을 말하거나 거짓말을 하며 게임에 참여합니다.

---

## 2. 핵심 컨셉

```text
사람과 AI가 같은 게임방에서 함께 플레이하는 하이브리드 마피아 게임

- 사람끼리만 플레이 가능
- 사람 + AI 혼합 플레이 가능
- 사용자 1명 + AI 다수 플레이 가능
- 부족한 인원은 AI로 자동 충원 가능
- 마피아와 직업은 사람/AI 구분 없이 랜덤 배정
- AI는 한국어 LLM 기반으로 대화 및 추리 수행
```

## 3. 주요 특징

- Android / iOS 동시 지원
- 클라우드 서버 기반 게임 운영
- 사람과 AI가 동일한 플레이어 슬롯으로 참여
- 사람끼리만 플레이 가능
- AI 포함 게임 가능
- 부족 인원 AI 자동 충원 가능
- 마피아 랜덤 배정
- 다양한 직업 시스템
- LLM 기반 한국어 AI 대화
- 실시간 채팅 및 투표
- 게임 결과 및 추리 리포트 제공

## 4. 지원 게임 모드

### 4.1 사람끼리만 플레이

```
사용자 6명
AI 없음
```
일반적인 온라인 마피아 게임처럼 사람 유저끼리만 플레이하는 모드입니다.

### 4.2 사람 + AI 혼합 플레이

```
사용자 2~5명
부족 인원 AI 자동 충원
```
사람 유저가 부족해도 AI가 빈 자리를 채워 게임을 시작할 수 있습니다.

### 4.3 사용자 1명 + AI 플레이

```
사용자 1명
AI 5명
```
싱글 플레이 또는 연습 모드로 사용할 수 있습니다.

### 4.4 AI끼리 자동 시뮬레이션

```
AI 6명
관전 모드
```
AI들이 서로 토론하고 투표하는 과정을 관전할 수 있는 확장 기능입니다.

## 5. MVP 범위

초기 MVP는 아래 범위로 개발합니다.
```
사용자 1명 + AI 5명
직업: 마피아, 시민, 경찰, 의사
게임 흐름: 낮 토론 → 투표 → 밤 행동 → 결과 판정
AI는 LLM 기반으로 한국어 대화 수행
```

### MVP 주요 기능

| 기능         | 설명                           |
| ---------- | ---------------------------- |
| 게임 시작      | 사용자가 게임을 생성하고 AI 플레이어를 자동 배치 |
| AI 플레이어 생성 | 서로 다른 이름, 성격, 말투를 가진 AI 생성   |
| 직업 랜덤 배정   | 사람과 AI 구분 없이 직업 랜덤 배정        |
| 낮 토론       | 사용자와 AI가 채팅으로 토론             |
| AI 발언      | AI가 직업과 성격에 맞게 발언            |
| 투표         | 플레이어 중 의심 대상을 선택             |
| 밤 행동       | 마피아, 경찰, 의사 직업 행동 처리         |
| 결과 판정      | 시민 팀 또는 마피아 팀 승패 판정          |
| 게임 기록      | 대화, 투표, 결과 저장                |

## 7. 게임 진행 흐름

```
1. 사용자가 게임 시작
2. 서버가 AI 플레이어 자동 생성
3. 모든 플레이어에게 직업 랜덤 배정
4. 낮 토론 시작
5. 사용자와 AI가 채팅으로 대화
6. 투표 진행
7. 최다 득표자 처형
8. 밤 행동 진행
9. 마피아, 경찰, 의사 능력 처리
10. 생존자 상태 갱신
11. 승패 조건 확인
12. 게임 종료 또는 다음 날 진행
```

## 8. 플레이어 구조

사람과 AI는 모두 동일한 Player 구조로 관리합니다.

```
Game Room
 ├─ Player 1: Human
 ├─ Player 2: Human
 ├─ Player 3: AI
 ├─ Player 4: AI
 ├─ Player 5: Human
 └─ Player 6: AI
 ```

중요한 설계 원칙:
```
- 사람과 AI는 모두 game_players 테이블에서 관리
- AI 여부는 is_ai 값으로 구분
- 직업 배정은 사람/AI 구분 없이 랜덤
- 채팅, 투표, 사망, 밤 행동은 동일한 게임 룰 적용
```

## 9. 직업 랜덤 배정 예시

6인 게임 기준:
```
총 인원: 6명

직업 구성:
- 마피아 1명
- 경찰 1명
- 의사 1명
- 시민 3명
```
즉, AI도 마피아가 될 수 있고 사람도 마피아가 될 수 있습니다.

## 10. 매칭 방식

**빠른 시작**

```
1. 사용자가 빠른 시작 클릭
2. 서버가 대기열에 사용자 등록
3. 같은 조건의 사용자들을 매칭
4. 일정 시간 안에 인원이 부족하면 AI로 충원
5. 게임 시작
```

예시 :
```
목표 인원: 6명
대기 사용자: 3명
부족 인원: 3명

결과:
사용자 3명 + AI 3명으로 게임 시작
```

## 12. AI 설계 원칙

LLM이 게임의 실제 상태를 직접 결정하면 안 됩니다.

게임의 진실과 상태는 서버가 관리하고, LLM은 AI 플레이어의 발언과 판단을 생성하는 역할만 담당합니다.

**서버가 관리하는 정보**

```
- 누가 마피아인지
- 누가 시민인지
- 누가 경찰인지
- 누가 의사인지
- 누가 생존 중인지
- 누가 사망했는지
- 누가 누구에게 투표했는지
- 밤 행동 결과
- 승패 조건
```

**LLM이 담당하는 정보**

```
- AI 플레이어의 말투
- 직업별 발언
- 거짓말
- 방어 발언
- 의심 표현
- 질문 답변
- 추리 발언
```

## 14. AI 직업별 행동 방식

**마피아 AI**

```
- 본인이 마피아라는 사실을 숨김
- 시민처럼 행동
- 다른 사람을 자연스럽게 의심
- 의심받으면 논리적으로 반박
- 너무 완벽하게 말하지 않음
```

**시민 AI**

```
- 대화 내용을 바탕으로 추리
- 의심되는 사람을 언급
- 다른 플레이어의 발언 모순을 지적
- 자신의 생존을 위해 설득
```

**경찰 AI**

```
- 밤에 특정 플레이어를 조사
- 조사 결과를 공개할지 숨길지 판단
- 마피아를 찾으면 설득 시도
```

**의사 AI**

```
- 밤에 특정 플레이어를 보호
- 본인의 정체를 숨김
- 중요해 보이는 플레이어를 보호
```

## 24. 개발 단계

**1단계: 게임 룰 엔진 구현**
```
- 게임 생성
- 플레이어 생성
- 직업 랜덤 배정
- 생존/사망 처리
- 승패 판정
```

**2단계: 사용자 1명 + AI 5명 모드 구현**
```
- AI 프로필 생성
- AI 자동 입장
- AI 발언 생성
- 사용자 채팅 입력
```

**3단계: 모바일 UI 구현**
```
- 홈 화면
- 게임 설정 화면
- 게임방 화면
- 채팅 화면
- 투표 화면
- 결과 화면
```

**4단계: WebSocket 적용**
```
- 실시간 채팅
- 투표 상태 동기화
- 게임 단계 변경 알림
- AI 발언 실시간 출력
```

**5단계: 사람 + AI 혼합 매칭 구현**

- 랜덤 매칭 큐

## 25. 최종 MVP 정의

```
AI 마피아 수사대 MVP

사용자는 모바일 앱에서 게임방을 생성하거나 빠른 시작을 선택할 수 있다.
게임은 사람끼리만 진행할 수도 있고, AI를 포함해 진행할 수도 있다.
방 설정에서 AI 포함 여부와 부족 인원 자동 충원 여부를 선택할 수 있다.
게임 시작 시 서버는 모든 참가자에게 직업을 랜덤으로 배정한다.
마피아는 사람 또는 AI 중 랜덤으로 정해진다.
AI 플레이어는 자신의 직업, 성격, 비밀 정보에 따라 한국어로 대화한다.
사용자와 AI는 낮 토론, 밤 행동, 투표 과정을 동일하게 수행한다.
최종적으로 시민 팀 또는 마피아 팀의 승패를 서버가 판정한다.
```

## 26. 프로젝트 목표

이 프로젝트의 목표는 단순한 마피아 게임이 아니라,<br>LLM 기반 AI 캐릭터가 사람처럼 추리하고 거짓말하며 대화하는 추리형 소셜 게임을 만드는 것입니다.

핵심 차별점은 다음과 같습니다.

```
1. 사람과 AI가 함께 플레이하는 하이브리드 마피아 게임
2. AI가 실제 플레이어처럼 발언, 거짓말, 방어, 추리 수행
3. 사람이 부족해도 AI가 자동 충원되어 게임 진행 가능
4. 매 판 다른 AI 성격과 대화 흐름 제공
5. 게임 종료 후 AI 기반 수사 리포트 제공
```
